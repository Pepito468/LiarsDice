# LiarsDice
Liar's Dice game made in Java

## Steps to play
* run Server
``` java -jar Server.jar ```
* run Client
```java -jar Client.jar username server_addres ```

The game starts when 3 players are in the lobby.
## How to play
Take turns and for every turn guess with the number of dices and the symbol (ex. "Three twos" = *3 2* ). If you think the last player lied, write *LIAR* and see who is right.
